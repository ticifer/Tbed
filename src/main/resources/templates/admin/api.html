功 能 接 口
请求方式 POST
请求地址

请求参数：

参数名 参数类型 必 填 参数说明
file file 是 图像文件
email String 是 用户邮箱
pass String 是 用户密码

返回参数：

参数名称 参数类型 参数说明
code Number 状态码，成功返回200，失败返回负数
msg String 提示信息
data Array|Object 图片上传后的信息

参数实例：

            //成功实例：
{
    "code": 200,
    "msg": "success",
    "data": [
        {
            "Imgname": "bloglogo.png",
            "Imgurl": "https://cdn.wwery.com/user/5dbfa0807100052.png"
        }
    ]
}

        
        //失败实例：
{
        "code": 200,
        "msg": "success",
        "data": [
        {
        "Imgname": "Tbed.zip",
        "Imgurl": "文件超出系统设定大小，不得超过"
        }
    ]
}

上传头需要是Content-Type: multipart/form-data;
